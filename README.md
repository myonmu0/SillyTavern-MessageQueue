# Message Queue
Send message to queue while generation, now you don't have to wait until generation end to send next messages. <br />
[YouTube Demo](https://www.youtube.com/watch?v=ux4G4DJJOJs)<br /><br />
*Screen shot*
![example](https://github.com/myonmu0/SillyTavern-MessageQueue/blob/main/demo.png)

## Install
SillyTavern > Extensions > Install extension > copy&paste this github url and install.

## Usage
- Enable/Disable on Extensions tab
- Press SHIFT + DEL on chat box to delete last queue, hold to delete all.
- How to use Split:
   1. Extensions > Message Queue > Check Split and Confirm/change the Delimiter.
   2. If you send this, those 3 messages will be queued and send one by one.
  ```
  Message1|||
  Message2|||
  Message3
  ```

## Limitation 
- If you switch browser tab and hide SillyTavern tab, queue text sending will pause until tab are visible again. Switching workspaces(Virtual Desktop) may work fine.


**Enjoy!**
